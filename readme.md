# RetailManager

Sistema de gerenciamento de varejo desenvolvido como **projeto acadêmico** para o curso de Ciência da Computação.

## Sobre o Projeto
Esta aplicação foi criada para aplicar conceitos fundamentais de Engenharia de Software e Programação, focando em:
* **Orientação a Objetos** (Herança, Polimorfismo e Encapsulamento).
* **Arquitetura em Camadas** (Separação entre Modelo, Visão e Persistência).
* **Padrão DAO** (Data Access Object) para comunicação com banco de dados.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** MySQL
* **Modelagem:** SQL e Diagramas ER

## Funcionalidades Principais
* Gestão de Vendas (PDV) e emissão de relatórios.
* Controle de Estoque de produtos.
* Cadastro e gerenciamento de Clientes, Funcionários e Fornecedores.
* Cálculo de comissões e métricas de vendas.
